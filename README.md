## вңҰ What is Muse.?

**Muse.** is a modern, responsive, multi-page music player built entirely with vanilla HTML, CSS, and JavaScript вҖ” no frameworks, no dependencies. It replicates the premium feel of native apps like Apple Music, featuring a dynamic glassmorphism UI, a fully functional audio engine, and fluid transitions between pages.

> Built as a Web Development course project by **Arjun Sheoran** & **Atharv Bains**.

---

## вңҰ Feature Highlights

| Feature | Description |
|---|---|
| рҹҺө **Dynamic Player** | Full audio playback with play/pause, skip, seek, and volume control |
| рҹҢҖ **Rotating Album Art** | Vinyl-style spinning artwork during active playback |
| рҹҺЁ **Adaptive Backgrounds** | Blurred, color-matched backdrop pulled from the current track's album art |
| рҹ”Җ **Random & Repeat** | Shuffle mode and single-track repeat toggle |
| рҹ“ҡ **Searchable Library** | 20-track library with live search/filter |
| рҹ—ӮпёҸ **Curated Playlists** | 4 hand-picked playlists: Pop Hits, Retro Rewind, Late Night Vibes, High Energy |
| рҹҸ  **Dynamic Home Feed** | Hero banner, horizontal scroll cards, and quick-link navigation |
| рҹ”җ **Auth Flow** | Simulated Login & Sign Up with localStorage state |
| рҹ“ұ **Fully Responsive** | Optimized for both mobile (app view) and desktop (dashboard view) |

## вңҰ Design Philosophy

Muse. draws inspiration from **Apple Music's** dark, premium interface language вҖ” but pushes it further with:

- **Glassmorphism** вҖ” frosted-glass cards, blurred backdrops, layered transparency
- **Dynamic theming** вҖ” backgrounds shift color with each track change
- **Fluid motion** вҖ” vinyl rotation, hover lifts, smooth transitions
- **Intentional typography** вҖ” hierarchy-first, system font stack for native feel
