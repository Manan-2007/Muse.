Project: Muse. - Web Based Music Player
Developers: Arjun Sheron, Athrav Bains
Course: Web Development (HTML/CSS/JS)

========================================================================
PROJECT OVERVIEW
========================================================================
Muse. is a responsive, multi-page music player application designed to 
mimic the premium experience of native apps like Apple Music. It features 
a modern dark UI, glassmorphism effects, and persistent playback state logic.

KEY FEATURES
- Authentication Flow: Simulated Login/Signup pages with welcome alerts.
- Dynamic Dashboard: Home page with horizontal scrolling and quick links.
- Playlists: Curated collections (Pop, Retro, Vibes) that launch specific mixes.
- Advanced Player: 
  * Dynamic blurred background that adapts to album art.
  * Custom styled range sliders for Seek and Volume.
  * Rotating album art animation during playback.
- Library Management: Searchable song list with duration data.
- Responsive Design: Works on Desktop (dashboard view) and Mobile (app view).

========================================================================
TECH STACK
========================================================================
- HTML5 (Semantic Structure)
- CSS3 (Flexbox, Grid, Variables, Animations, Backdrop Filters)
- JavaScript (DOM Manipulation, Audio API, Event Handling)
